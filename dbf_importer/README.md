# DBF Importer

Отдельное переносимое приложение для импорта данных из DBF файлов в систему управления строительством.

## Структура приложения

```
dbf_importer/
├── README.md                 # Этот файл
├── requirements.txt          # Зависимости Python
├── config/
│   ├── __init__.py
│   └── settings.py           # Настройки приложения
├── core/
│   ├── __init__.py
│   ├── dbf_reader.py         # Модуль чтения DBF файлов
│   ├── database.py           # Модуль работы с базой данных
│   └── importer.py           # Основная логика импорта
├── ui/
│   ├── __init__.py
│   ├── main_window.py        # Главное окно приложения
│   ├── import_dialog.py      # Диалог импорта
│   └── progress_dialog.py    # Диалог прогресса
├── docs/
│   ├── user_guide.md         # Руководство пользователя
│   └── dbf_structure.md      # Описание структуры DBF файлов
└── main.py                   # Точка входа в приложение
```

## Функциональность

- Чтение DBF файлов из системы 1С
- Импорт данных в базу данных SQLite
- Отображение прогресса импорта
- Обработка ошибок и валидация данных
- Поддержка различных сущностей (Номенклатура, Работы, Материалы, Элементы затрат)

## Установка и запуск

1. Установить зависимости:
   ```
   pip install -r requirements.txt
   ```

2. Запустить приложение:
   ```
   python main.py
   ```

## Требования

- Python 3.8+
- PyQt6
- dbfread
- SQLAlchemy
