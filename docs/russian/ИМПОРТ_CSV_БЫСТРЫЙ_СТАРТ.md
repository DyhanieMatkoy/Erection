# Импорт работ из CSV - Быстрый старт

## Веб-интерфейс (рекомендуется)

1. Откройте веб-интерфейс
2. **Справочники** → **Работы**
3. Кнопка **"Импорт CSV"**
4. Выберите файл `умелец.csv`
5. **"Импортировать"**

✅ Готово! 228 работ импортировано.

## Командная строка

```bash
# Импорт
import_works.bat умелец.csv

# Просмотр
view_works.bat
```

## Формат файла

```csv
Тип работ;Наименование работы;Цена;Единица измерения
Электромонтажные работы;монтаж розетки;500;руб.
```

- Разделитель: **;** (точка с запятой)
- Кодировка: **UTF-8**
