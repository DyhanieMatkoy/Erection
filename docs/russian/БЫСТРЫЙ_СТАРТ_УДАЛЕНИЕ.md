# Быстрый старт: Удаление помеченных элементов

## Что добавлено?

В web UI добавлена возможность удалять помеченные элементы справочников.

## Как использовать?

### Вариант 1: Пометка и удаление через UI

1. **Откройте справочник** (например, Работы)
2. **Выберите элементы** для удаления (чекбоксы слева)
3. **Нажмите "Пометить на удаление"** - элементы будут помечены красным индикатором
4. **Нажмите "Удалить помеченные"** (красная кнопка вверху) - элементы будут удалены навсегда

### Вариант 2: Массовое удаление через CSV

1. **Создайте CSV файл** со списком работ для удаления:
   ```csv
   Тип работ;Наименование работы;Цена;Единица измерения
   ;Работа для удаления 1;;
   ;Работа для удаления 2;;
   ```

2. **Откройте "Работы"** → **"Импорт CSV"**

3. **Выберите файл** и включите **"Режим удаления"**

4. **Нажмите "Импортировать"** - все работы с указанными наименованиями будут помечены

5. **Нажмите "Удалить помеченные"** для окончательного удаления

## Важно!

⚠️ **Физическое удаление необратимо!** Восстановление возможно только из резервной копии БД.

⚠️ **Требуются права администратора** для физического удаления.

## Что изменилось?

### Backend (API)
- Добавлены endpoints для физического удаления помеченных элементов
- Режим удаления в импорте CSV уже был, теперь используется в UI

### Frontend (Web UI)
- Кнопка "Удалить помеченные" появляется автоматически при наличии помеченных элементов
- Чекбокс "Режим удаления" в модальном окне импорта CSV
- Изменена надпись кнопки: "Удалить" → "Пометить на удаление"
