# Инструкция: Создание Ежедневного отчёта

## Что исправлено

### 1. Добавлена кнопка создания отчёта в списке смет

Теперь в форме "Документы → Сметы" есть кнопка **"Создать Ежедневный отчёт"**, которая позволяет быстро создать новый ежедневный отчёт на основе выбранной сметы.

### 2. Исправлен выбор сметы и бригадира

В непроведенном ежедневном отчёте теперь можно:
- Выбрать смету через кнопку "..." (открывается список смет)
- Выбрать бригадира через кнопку "..." (открывается справочник физических лиц)

## Как создать ежедневный отчёт

### Способ 1: Из списка смет (рекомендуется)

1. Откройте **Документы → Сметы**
2. Выберите нужную смету в списке (кликните на строку)
3. Нажмите кнопку **"Создать Ежедневный отчёт"**
4. Откроется форма нового отчёта с уже выбранной сметой
5. Выберите бригадира:
   - Нажмите кнопку "..." рядом с полем "Бригадир"
   - В открывшемся окне выберите нужного человека
   - Нажмите "Выбрать" (или Ctrl+Enter)
6. Нажмите **"Заполнить из сметы"** для добавления работ
7. Выберите нужные строки и нажмите "OK"
8. Заполните фактические трудозатраты и исполнителей
9. Нажмите **"Сохранить и закрыть"** (Ctrl+Shift+S)

### Способ 2: Через контекстное меню

1. Откройте **Документы → Сметы**
2. Кликните правой кнопкой мыши на нужной смете
3. Выберите **"Создать Ежедневный отчет"** из меню
4. Далее действуйте как в Способе 1 (шаги 4-9)

### Способ 3: Создание с нуля

1. Откройте **Документы → Ежедневные отчёты**
2. Нажмите **"Создать"** (Insert или F9)
3. Выберите смету:
   - Нажмите кнопку "..." рядом с полем "Смета"
   - В открывшемся окне выберите нужную смету
   - Нажмите "Выбрать"
4. Выберите бригадира (см. Способ 1, шаг 5)
5. Далее действуйте как в Способе 1 (шаги 6-9)

## Выбор исполнителей

Для каждой работы в отчёте можно указать исполнителей:

1. Дважды кликните на ячейку в колонке "Исполнители"
   - Или нажмите F4, находясь в этой колонке
2. В открывшемся окне отметьте галочками нужных исполнителей
3. Нажмите "Выбрать"

## Проведение отчёта

После заполнения всех данных:

1. Нажмите **"Провести"** (Ctrl+K)
2. Отчёт будет проведён и данные попадут в регистр выполнения работ
3. После проведения отчёт нельзя редактировать
4. Для изменений нужно сначала отменить проведение

## Печать отчёта

1. Откройте нужный отчёт
2. Нажмите **"Печать"** (Ctrl+P)
3. Выберите формат печати в настройках (PDF или Excel)
4. Отчёт откроется в программе по умолчанию

## Горячие клавиши

- **Ctrl+S** - Сохранить
- **Ctrl+Shift+S** - Сохранить и закрыть
- **Ctrl+K** - Провести документ
- **Ctrl+P** - Печать
- **F4** - Выбрать исполнителей (в колонке "Исполнители")
- **Esc** - Закрыть форму

## Примечания

- Смету и бригадира можно выбрать только в непроведенном отчёте
- После проведения отчёта эти поля становятся недоступными для редактирования
- Для изменения проведенного отчёта нужно сначала отменить проведение
- Бригадиры видят только свои отчёты (по умолчанию)
- Администраторы и мастера видят все отчёты
