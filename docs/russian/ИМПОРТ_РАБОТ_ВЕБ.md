# Импорт работ из CSV в веб-интерфейсе

## Быстрый старт

1. Откройте **Справочники** → **Работы**
2. Нажмите кнопку **"Импорт CSV"** в правом верхнем углу
3. Выберите CSV файл
4. (Опционально) Выберите родительскую группу
5. Нажмите **"Импортировать"**

## Формат CSV файла

```csv
Тип работ;Наименование работы;Цена;Единица измерения
Электромонтажные работы;монтаж розетки (220 В);500;руб.
Сантехнические работы;установка унитаза;1500;руб.
```

**Важно:**
- Разделитель: точка с запятой (;)
- Кодировка: UTF-8
- Первая строка - заголовки

## Параметры импорта

### Родительская группа
- **Не выбрана**: работы группируются по полю "Тип работ"
- **Выбрана**: все работы добавляются в эту группу

### Пропускать существующие работы
- **Включено** (по умолчанию): существующие работы не изменяются
- **Выключено**: существующие работы обновляются

## Пример файла

Используйте файл `умелец.csv` в корне проекта как пример.

## Результат импорта

После импорта вы увидите:
- ✅ Количество добавленных работ
- ⏭️ Количество пропущенных работ
- ❌ Список ошибок (если есть)

Таблица работ автоматически обновится.
