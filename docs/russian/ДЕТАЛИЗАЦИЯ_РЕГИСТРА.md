# Детализация регистра выполнения работ

## Что добавлено

В отчете по выполненным работам теперь можно открыть детализацию движений и перейти к исходным документам (сметам и ежедневным отчетам).

## Как использовать

### Desktop-приложение

1. Откройте отчет по выполненным работам
2. Дважды кликните по любой строке в таблице
3. Откроется окно с детализацией всех движений
4. Дважды кликните по движению, чтобы открыть документ (смету или daily report)

### Веб-клиент

1. Откройте раздел "Регистры" → "Выполнение работ"
2. Установите фильтры (период обязателен) и нажмите "Применить"
3. Дважды кликните по любой строке в таблице
4. Откроется модальное окно с детализацией движений
5. Дважды кликните по движению, чтобы открыть документ

## Что показывается в детализации

- Тип документа (Смета или Ежедневный отчет)
- Номер документа или дата
- Дата движения
- Приход: количество и сумма
- Расход: количество и сумма

## Примечания

- Детализация учитывает все активные фильтры (объект, смета, работа)
- При открытии документа из детализации вы переходите к форме редактирования документа
- В desktop-приложении документ открывается в новом окне
- В веб-клиенте происходит переход на страницу документа
