# Загрузка сметы из Excel

## Описание

Добавлена возможность загрузки данных сметы из файла Excel непосредственно в форме документа сметы.

## Использование

1. Откройте форму документа сметы (новую или существующую)
2. Нажмите кнопку **"Загрузить из Excel"**
3. Выберите файл Excel (*.xlsx или *.xls) с данными сметы
4. Система автоматически импортирует:
   - Номер сметы
   - Заказчика
   - Объект
   - Подрядчика
   - Табличную часть с работами
   - Количество, цены, трудозатраты

5. После импорта проверьте данные и нажмите **"Сохранить"**

## Важные замечания

- **Проведенные документы**: Нельзя загружать данные в проведенный документ. Сначала отмените проведение.
- **Несохраненные изменения**: Если в форме есть несохраненные изменения, система предупредит вас перед загрузкой.
- **Замена данных**: При загрузке все текущие данные в форме будут заменены данными из Excel.
- **Автоматическое создание**: Если заказчик, объект, подрядчик или работы не найдены в базе, они будут созданы автоматически.

## Формат Excel файла

Система поддерживает стандартный формат локальных смет АРСД:
- Заголовок с информацией о заказчике и подрядчике
- Таблица с колонками: №, Код, Наименование работ, Ед.изм., Количество, Цена, Трудозатраты
- Автоматическое определение начала табличной части

## Пример

После успешного импорта система покажет сообщение:
```
Импортировано строк: 25
Итого сумма: 1250000.00
Итого трудозатраты: 450.50
```

## Расположение кнопки

Кнопка **"Загрузить из Excel"** находится в нижней части формы документа сметы, рядом с кнопкой "Печать".
