# Инструкция: Сдельная форма по бригаде

## Назначение

Отчет "Сдельная форма по бригаде" предназначен для формирования детального отчета о выполненных работах с разбивкой по датам в формате Excel. Отчет показывает плановые и фактические объемы работ, их стоимость, а также детализацию выполнения по каждой дате.

## Как сформировать отчет

### Шаг 1: Открыть форму отчета

1. Запустите программу
2. В главном меню выберите **"Отчеты"** → **"Выполнение работ"**

### Шаг 2: Установить параметры

В форме отчета установите необходимые параметры:

#### Обязательные параметры:

- **Период с ... по ...** - укажите начальную и конечную даты периода отчета

#### Необязательные фильтры:

- **Объект** - выберите конкретный объект строительства (или оставьте "Все объекты")
- **Смета** - выберите конкретную смету (или оставьте "Все сметы")
- **Работа** - выберите конкретный вид работ (или оставьте "Все работы")
- **Исполнитель** - выберите конкретного исполнителя/бригадира (или оставьте "Все исполнители")

### Шаг 3: Сформировать отчет

1. Нажмите кнопку **"Сформировать"** - данные отобразятся в таблице
2. Нажмите кнопку **"Сдельная форма по бригаде"** - откроется диалог сохранения файла
3. Выберите место сохранения и имя файла
4. Нажмите **"Сохранить"**

### Шаг 4: Открыть отчет

Откройте сохраненный файл в Microsoft Excel или другой программе для работы с таблицами.

## Структура отчета

### Заголовок

В верхней части отчета указывается:
- Название проекта (объекта)
- Исполнитель (бригадир)
- Период отчета

### Основная таблица

Таблица содержит следующие колонки:

1. **№ п/п** - порядковый номер работы
2. **Наименование работ** - название выполняемой работы
3. **Исполнитель** - ФИО исполнителя (бригадира)
4. **Ед. изм.** - единица измерения (м2, м, шт и т.д.)
5. **Стоимость на ед.** - цена за единицу работы

#### Объём по проекту:
- **План** - плановое количество работ
- **Факт** - фактически выполненное количество
- **+/-** - отклонение (факт минус план)
- **%** - процент выполнения

#### Стоимость по проекту:
- **План** - плановая стоимость
- **Факт** - фактическая стоимость
- **+/-** - отклонение (факт минус план)
- **%** - процент выполнения

#### Даты выполнения:

Для каждой даты, когда выполнялись работы, создаются две колонки:
- **Факт объёмы** - количество выполненных работ в эту дату
- **Факт стоимость** - стоимость выполненных работ в эту дату

### Итоговая строка

В конце таблицы выводится строка "Итого" с суммами по всем работам.

## Примеры использования

### Пример 1: Отчет по всем работам за месяц

1. Период: с 01.11.2025 по 30.11.2025
2. Все остальные фильтры: "Все..."
3. Результат: отчет по всем работам, выполненным в ноябре 2025 года

### Пример 2: Отчет по конкретному объекту

1. Период: с 01.01.2025 по 31.12.2025
2. Объект: "Торговый центр 'Центральный'"
3. Остальные фильтры: "Все..."
4. Результат: отчет по всем работам на выбранном объекте за весь год

### Пример 3: Отчет по конкретному исполнителю

1. Период: с 01.10.2025 по 31.10.2025
2. Исполнитель: "Петров Петр Петрович"
3. Остальные фильтры: "Все..."
4. Результат: отчет по всем работам, выполненным бригадой Петрова в октябре

### Пример 4: Отчет по конкретной работе

1. Период: с 01.01.2025 по 31.12.2025
2. Работа: "Монтаж кабель-канала"
3. Остальные фильтры: "Все..."
4. Результат: отчет только по монтажу кабель-канала за весь год

## Особенности

### Цветовое оформление

- **Заголовки колонок** - голубой фон, жирный шрифт
- **Итоговая строка** - жёлтый фон, жирный шрифт
- **Данные** - белый фон, обычный шрифт

### Форматирование чисел

- Количество и стоимость - с двумя знаками после запятой (например: 123.45)
- Проценты - с двумя знаками и символом % (например: 85.50%)

### Автоматические расчеты

Отчет автоматически рассчитывает:
- Отклонения (факт минус план)
- Проценты выполнения (факт / план × 100%)
- Итоговые суммы по всем работам

## Часто задаваемые вопросы

**В: Почему в отчете нет некоторых работ?**

О: В отчет попадают только те работы, по которым есть плановые данные в сметах. Если работа не включена в смету, она не появится в отчете.

**В: Откуда берутся данные об исполнителях?**

О: Данные об исполнителях берутся из ежедневных отчетов (поле "Бригадир"). Если работа не связана с ежедневным отчетом, в колонке "Исполнитель" будет указано "Все исполнители".

**В: Почему некоторые даты отсутствуют в отчете?**

О: В отчете отображаются только те даты, в которые фактически выполнялись работы. Если в какой-то день работы не велись, колонка для этой даты не создается.

**В: Можно ли изменить формат отчета?**

О: Да, после сохранения файла вы можете открыть его в Excel и изменить форматирование, добавить графики, изменить ширину колонок и т.д.

**В: Как получить отчет за несколько месяцев?**

О: Просто укажите нужный период в параметрах. Например, с 01.01.2025 по 31.03.2025 для отчета за первый квартал.

## Техническая поддержка

Если у вас возникли проблемы с формированием отчета:

1. Проверьте, что в базе данных есть данные за указанный период
2. Убедитесь, что выбранные фильтры не слишком строгие
3. Попробуйте сформировать отчет без фильтров
4. Проверьте, что у вас установлена актуальная версия программы

Для получения дополнительной помощи обратитесь к администратору системы.
