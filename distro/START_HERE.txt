QUICK START:

1. Install Prerequisites (in order):
   - prerequisites\python-3.11.x-amd64.exe  (Check "Add to PATH")
   - prerequisites\node-v20.x.x-x64.msi
   - prerequisites\VC_redist.x64.exe

2. Run Installer:
   - Double-click: create_installer.bat
   - Follow on-screen instructions

3. Launch Application:
   - Desktop shortcut: "Construction Time Management"
   - Or run: C:\ConstructionTimeManagement\run.bat

4. Login:
   - Username: admin
   - Password: admin
   - CHANGE PASSWORD AFTER FIRST LOGIN!
