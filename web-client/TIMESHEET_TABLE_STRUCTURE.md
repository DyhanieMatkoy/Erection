# Структура таблицы табеля с прокруткой

## Визуальная схема

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                            Табель (Timesheet)                                │
└─────────────────────────────────────────────────────────────────────────────┘

┌──────────────────┬──────────────────────────────────────┬─────────────────────┐
│  ФИКСИРОВАННАЯ   │      ПРОКРУЧИВАЕМАЯ СЕКЦИЯ          │   ФИКСИРОВАННАЯ     │
│   ЛЕВАЯ ЧАСТЬ    │     (overflow-x: auto)               │   ПРАВАЯ ЧАСТЬ      │
├──────────────────┼──────────────────────────────────────┼─────────────────────┤
│ Сотрудник│Ставка │ 1│ 2│ 3│...│29│30│31                │ Итого│Сумма│Действия│
├──────────────────┼──────────────────────────────────────┼─────────────────────┤
│ Иванов И.│ 150.00│ 8│ 8│ 8│...│ 8│ 8│ 0                │ 240.0│36000│ Удалить │
│ Петров П.│ 180.00│ 8│ 8│ 8│...│ 8│ 8│ 0                │ 240.0│43200│ Удалить │
│ Сидоров С│ 200.00│ 8│ 8│ 8│...│ 8│ 8│ 0                │ 240.0│48000│ Удалить │
├──────────────────┼──────────────────────────────────────┼─────────────────────┤
│ Итого:   │       │24│24│24│...│24│24│ 0                │ 720.0│127200│         │
└──────────────────┴──────────────────────────────────────┴─────────────────────┘
         ↑                        ↑                                ↑
    Не двигается          Прокручивается →                  Не двигается
```

## Технические детали

### HTML структура

```html
<div class="border border-gray-200 rounded-lg overflow-hidden">
  <div class="flex">
    
    <!-- 1. Левая фиксированная секция -->
    <div class="flex-shrink-0 border-r border-gray-200">
      <table>
        <thead>
          <tr>
            <th>Сотрудник</th>
            <th>Ставка</th>
          </tr>
        </thead>
        <tbody>
          <!-- Строки сотрудников -->
        </tbody>
      </table>
    </div>

    <!-- 2. Центральная прокручиваемая секция -->
    <div class="flex-1 overflow-x-auto">
      <table>
        <thead>
          <tr>
            <th v-for="day in daysInMonth">{{ day }}</th>
          </tr>
        </thead>
        <tbody>
          <!-- Колонки дней для каждого сотрудника -->
        </tbody>
      </table>
    </div>

    <!-- 3. Правая фиксированная секция -->
    <div class="flex-shrink-0 border-l border-gray-200">
      <table>
        <thead>
          <tr>
            <th>Итого</th>
            <th>Сумма</th>
            <th>Действия</th>
          </tr>
        </thead>
        <tbody>
          <!-- Итоговые колонки -->
        </tbody>
      </table>
    </div>

  </div>
</div>
```

### CSS классы

- `flex` - создает flexbox контейнер
- `flex-shrink-0` - запрещает сжатие фиксированных секций
- `flex-1` - позволяет центральной секции занимать оставшееся пространство
- `overflow-x-auto` - добавляет горизонтальную прокрутку
- `border-r` / `border-l` - границы между секциями

### Цветовая схема

- **Выходные дни**: `bg-red-50 text-red-700`
- **Итоговые колонки**: `bg-blue-50` (заголовки), `bg-blue-100` (итоги)
- **Строка итогов**: `bg-gray-100` (общая строка), `bg-blue-100` (итоговые ячейки)

## Преимущества новой структуры

1. ✅ **Всегда видны имена сотрудников** - даже при прокрутке дней
2. ✅ **Всегда видны итоги** - не нужно прокручивать до конца
3. ✅ **Удобная навигация** - можно быстро найти нужный день
4. ✅ **Адаптивность** - работает на любых размерах экрана
5. ✅ **Синхронизация высоты строк** - все три таблицы выровнены

## Поведение на разных экранах

### Широкий экран (> 1200px)
- Все колонки видны без прокрутки
- Таблица занимает всю ширину

### Средний экран (768px - 1200px)
- Появляется горизонтальная прокрутка для дней
- Фиксированные колонки остаются видимыми

### Узкий экран (< 768px)
- Активная горизонтальная прокрутка
- Фиксированные колонки занимают минимальную ширину
- Дни прокручиваются в центральной части

## Альтернативные подходы (не использованы)

1. **position: sticky** - не работает внутри overflow контейнеров
2. **Одна таблица с фиксированными колонками** - сложная синхронизация прокрутки
3. **JavaScript прокрутка** - избыточная сложность, проблемы с производительностью
