# Руководство по шрифтам в печатных формах

## Автоматический выбор шрифтов
Система автоматически выбирает подходящий шрифт с поддержкой кириллицы при генерации PDF.

## Поддерживаемые шрифты

### Windows (автоматически)
- Arial ✓ (рекомендуется)
- Times New Roman ✓
- Calibri ✓
- Verdana ✓

### Пользовательские шрифты
Вы можете добавить свои шрифты в папку `fonts/`:
- `fonts/DejaVuSans.ttf`
- `fonts/DejaVuSans-Bold.ttf`

## Проверка текущего шрифта
```python
from src.services.print_form_generator import PrintFormGenerator

generator = PrintFormGenerator()
print(f"Используется шрифт: {generator.font_name}")
```

## Устранение проблем

### Квадратики вместо букв
Если вы видите квадратики вместо русских букв:
1. Убедитесь, что используется не Helvetica
2. Проверьте наличие Arial в системе
3. Добавьте DejaVu Sans в папку `fonts/`

### Проверка доступных шрифтов
```bash
python test_cyrillic_fonts.py
```

Этот скрипт создаст тестовый PDF с кириллицей.

## Технические детали
- Шрифты встраиваются в PDF (размер ~85-90 KB)
- Поддержка полного набора кириллических символов
- Автоматический fallback на доступные шрифты
