# Реализация системы навигации

## Обзор

Реализована полноценная система навигации для главного окна приложения в стиле 1С 8.3.

## Реализованные функции

### 1. Быстрая навигация (Ctrl+K)
- Диалоговое окно для быстрого доступа к формам
- Поиск по названию формы с фильтрацией в реальном времени
- Навигация стрелками и выбор через Enter
- Доступ ко всем основным формам системы

### 2. Управление окнами
- **Режим вкладок** (по умолчанию) - все формы открываются во вкладках
- **Режим каскада** - окна располагаются каскадом
- **Режим мозаики** - окна располагаются плиткой
- Переключение между режимами через меню "Окна"

### 3. Горячие клавиши
- `Ctrl+K` - Быстрая навигация
- `Ctrl+Tab` - Следующее окно
- `Ctrl+Shift+Tab` - Предыдущее окно
- `Ctrl+W` - Закрыть активное окно
- `Ctrl+Shift+W` - Закрыть все окна
- `Alt+F4` - Выход из приложения

### 4. История недавних форм
- Отслеживание последних 10 открытых форм
- Подменю "Недавние" в меню "Окна"
- Быстрый доступ к часто используемым формам

### 5. Меню "Окна"
- Каскадом - расположить окна каскадом
- Мозаикой - расположить окна плиткой
- Вкладки - переключение режима вкладок
- Закрыть активное окно
- Закрыть все окна
- Недавние - список недавно открытых форм

### 6. Улучшенное управление MDI
- Проверка на дублирование окон - если форма уже открыта, активируется существующее окно
- Вкладки с возможностью закрытия и перемещения
- Автоматическое отслеживание открытых форм

## Технические детали

### Класс QuickNavigationDialog
Диалоговое окно для быстрой навигации с:
- Полем поиска с автофильтрацией
- Списком доступных форм
- Поддержкой клавиатурной навигации

### Класс MainWindow
Расширен следующими методами:
- `show_quick_navigation()` - показать диалог быстрой навигации
- `add_to_recent()` - добавить форму в историю
- `update_recent_menu()` - обновить меню недавних форм
- `cascade_windows()` - расположить окна каскадом
- `tile_windows()` - расположить окна мозаикой
- `toggle_view_mode()` - переключить режим отображения
- `close_active_window()` - закрыть активное окно
- `close_all_windows()` - закрыть все окна
- `next_window()` - следующее окно
- `previous_window()` - предыдущее окно

### Структура данных
- `recent_forms` - deque с максимальным размером 10 для хранения истории

## Использование

### Быстрая навигация
1. Нажмите `Ctrl+K` или выберите "Файл" → "Быстрая навигация"
2. Начните вводить название формы
3. Используйте стрелки для выбора
4. Нажмите Enter для открытия

### Переключение между окнами
- `Ctrl+Tab` - переключение вперед
- `Ctrl+Shift+Tab` - переключение назад
- Или используйте вкладки мышью

### Управление расположением окон
1. Откройте меню "Окна"
2. Выберите нужный режим:
   - "Каскадом" - для каскадного расположения
   - "Мозаикой" - для плиточного расположения
   - "Вкладки" - для режима вкладок

### Недавние формы
1. Откройте меню "Окна" → "Недавние"
2. Выберите нужную форму из списка

## Соответствие требованиям

Реализация соответствует требованию 11.2 "Реализовать систему навигации" и обеспечивает:
- Удобную навигацию между формами
- Управление открытыми окнами
- Быстрый доступ к часто используемым формам
- Интуитивный интерфейс в стиле 1С 8.3
