# Система управления рабочим временем строительных бригад

Qt6 MVP приложение для управления строительными сметами, учета трудозатрат и формирования отчетности.

## Требования

- Qt 6.x
- CMake 3.16+
- C++17 компилятор
- SQLite (встроен в Qt)

## Сборка

### Windows

```bash
build.bat
```

### Linux/Mac

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Запуск

```bash
./build/ConstructionTimeManagement
```

## Функциональность

- Управление справочниками (Контрагенты, Объекты, Работы, Физлица, Организации)
- Создание и редактирование смет
- Учет ежедневных отчетов о выполненных работах
- **Печатные формы в формате АРСД** (смета, ежедневный отчет)
- Аналитические отчеты
- Разграничение прав доступа
- Горячие клавиши в стиле 1С 8.3

## Печатные формы

Реализованы печатные формы в формате АРСД:

- **Локальный сметный расчет** - печать сметы с разбивкой на единичные и полные объемы работ
- **Ежедневный отчет о выполненных работах** - отчет с плановыми и фактическими трудозатратами

Для печати используйте кнопку "Печать (Ctrl+P)" в формах документов.

**Поддержка кириллицы:** Система автоматически использует шрифты с поддержкой русского языка (Arial, Times New Roman и др.). Кириллические символы отображаются корректно.
