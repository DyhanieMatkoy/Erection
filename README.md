# Система управления рабочим временем строительных бригад

Приложение для управления строительными сметами, учета трудозатрат и формирования отчетности.
Включает в себя десктопное приложение (PyQt6) и веб-интерфейс (Vue 3 + FastAPI).

## Требования

- Python 3.10+
- Node.js 16+ (для разработки веб-клиента)

## Быстрый старт

### Запуск сервера (API + Веб)

```bash
start_server.bat
```
Сервер будет доступен по адресу: http://localhost:8000

### Запуск десктопного приложения

```bash
start_desktop.bat
```

## Учетные данные по умолчанию

- `admin` / `admin` (Администратор)
- `manager` / `manager` (Руководитель)
- `foreman` / `foreman` (Бригадир)

### Управление пользователями

Для сброса пароля администратора или управления пользователями:

```bash
python manage_users.py
```
или
```bash
python scripts/auth/reset_admin_password.py
```

## Структура проекта

```
src/            # Исходный код десктопного приложения (PyQt6)
web-client/     # Веб-интерфейс (Vue 3 + TypeScript)
api/            # REST API (FastAPI)
docs/           # Документация
```
